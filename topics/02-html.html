יסודות HTML

HTML הוא השלד של כל דף באינטרנט. בנושא הזה נבין מה זה HTML, איך בנויה תגית ואלמנט, איך מרכיבים מסמך שלם, ואיך הדפדפן קורא את הקוד והופך אותו לדף שרואים על המסך.

מה זה HTML ולמה הוא קיים

HTML הוא ראשי תיבות של HyperText Markup Language, כלומר "שפת סימון של היפר-טקסט". חשוב להבין: HTML היא לא שפת תכנות, היא שפת סימון. אנחנו לא כותבים בה תנאים ולולאות, אלא מתארים לדפדפן מה יש בדף: כאן כותרת, כאן פסקה, כאן תמונה, כאן קישור.

הדפדפן קורא את קובץ ה-HTML (שהוא טקסט רגיל בלבד) ומעבד אותו כדי לבנות את ממשק המשתמש שאנחנו רואים. המצגת מדגישה כמה עובדות חשובות:

  • HTML היא השפה הכי נפוצה בעולם ה-IT. כל אתר באינטרנט מבוסס עליה.
  • HTML היא פשוטה מאוד ללמידה.
  • HTML היא כמעט השפה היחידה שסלחנית לשגיאות (error tolerance). אם תשכח לסגור תגית, הדפדפן ינסה בכל זאת להציג משהו הגיוני ולא יקרוס.
  • קובץ ה-HTML עצמו הוא טקסט בלבד, אבל אחרי שהדפדפן מעבד אותו (rendering) הוא יכול להכיל תמונות, וידאו ואודיו.
💡 למה זה סלחני לשגיאות דווקא?

כי HTML נועד לכל אחד, גם למי שאינו מתכנת. אם דף היה קורס בגלל תו אחד חסר, האינטרנט לא היה נבנה כל כך מהר. המחיר הוא שקל לכתוב HTML "מבולגן" שעדיין עובד. לכן חשוב ללמוד לכתוב קוד נקי ומסודר.

תחביר: תגית, אלמנט ומאפיין

שלושה מושגים שחייבים להכיר. נפרק אותם אחד-אחד.

תגית (Tag)

תגית נבנית עם הסימנים < ו-> ויש לה שם. שם התגית מוגדר מראש על ידי השפה והוא בעצם פקודה לדפדפן. רוב התגיות מגיעות בזוג: תגית פתיחה ותגית סגירה (עם לוכסן). HTML אינה רגישה לאותיות גדולות או קטנות, אבל מקובל לכתוב באותיות קטנות.

<h1> היא תגית הפתיחה, </h1> היא תגית הסגירה, והמילה "שלום" היא התוכן.

אלמנט (Element)

אלמנט הוא הכל ביחד: תגית הפתיחה, תגית הסגירה, וכל מה שביניהן. אלמנט יכול להכיל אלמנטים אחרים בתוכו, וזה נקרא אלמנטים מקוננים (nested). יש גם אלמנטים ריקים שאין להם תגית סגירה, כמו <br> (מעבר שורה), <hr>, <img> ו-<input>. בנקודה הזו HTML שונה מ-XML: ב-XML כל תגית חייבת סגירה, וגם HTML עצמה אינה מבוססת על XML.

מאפיין (Attribute)

אלמנט יכול לכלול מאפיינים. מאפיין הוא זוג של שם וערך, והוא נמצא רק בתגית הפתיחה. המאפיינים מספרים לדפדפן איך להריץ או להציג את האלמנט. סדר המאפיינים לא משנה, וערך אפשר לעטוף במרכאות כפולות, בודדות, או להשאיר בלי מרכאות בכלל (זו לא שגיאה).

כאן ל-img יש שלושה מאפיינים: src (הנתיב לתמונה), alt (טקסט חלופי אם התמונה לא נטענת) ו-width (הרוחב בפיקסלים). ה-src יכול להיות שם קובץ מקומי או כתובת URL מלאה, והתמונה יכולה להיות בכל פורמט (gif, png, jpg ועוד), לא רק gif.

מבנה מסמך HTML

לכל דף HTML יש שלד קבוע. נכיר את האלמנטים המרכזיים:

  • <html> - אלמנט השורש, עוטף את כל המסמך.
  • <head> - מכיל מידע על הדף (כותרת, קידוד, קישורים לקבצי CSS). התוכן שלו לא מוצג ישירות בגוף הדף.
  • <title> - הכותרת שמופיעה בלשונית הדפדפן. נמצא בתוך ה-head.
  • <body> - הגוף. כאן נמצא כל התוכן שהמשתמש רואה.

הערות בקוד נכתבות כך: <!-- הערה -->. הדפדפן מתעלם מהן, והן נועדו למתכנתים.

איך הדפדפן קורא HTML ובונה את הדף

כותרות, פסקאות ומעברים

אלה האלמנטים הבסיסיים להצגת טקסט:

  • <h1> עד <h6> - שש רמות של כותרות. h1 הכי גדולה, h6 הכי קטנה.
  • <p> - פסקה. מתחילה בשורה חדשה.
  • <br> - מעבר שורה (אלמנט ריק, בלי תגית סגירה).
  • <hr> - קו אופקי שמפריד בין חלקים.

רשימות

שני סוגי רשימות עיקריים:

  • <ol> - רשימה ממוספרת (Ordered List). אפשר לשלוט בסוג המספור עם המאפיין type (למשל 1, a, A, i, I).
  • <ul> - רשימה לא ממוספרת (Unordered List), עם נקודות. המאפיין type יכול להיות disc, square או circle.
  • <li> - פריט ברשימה (List Item). נמצא בתוך ol או ul.

קישורים

אלמנט הקישור <a> (anchor) יוצר אזור שאפשר ללחוץ עליו כדי לעבור לדף אחר. המאפיין החשוב שלו הוא href (הכתובת). מאפיין target קובע איפה הקישור ייפתח, למשל _blank פותח בלשונית חדשה. אפשר גם לקפוץ למקום מסוים באותו דף בעזרת # ו-id.

div, span, וסמנטיקה מול עיצוב

למה בכלל להשתמש באלמנט מסוים? יש שתי סיבות:

  • כדי להוסיף משמעות (semantics). למשל, <h1> אומר "זו הכותרת הראשית", וזה עוזר למשתמש, לגוגל ולתוכנות קריאה.
  • כדי לשנות עיצוב (בעזרת CSS).

שני אלמנטים חשובים שאין להם משמעות מיוחדת, והם קיימים בעיקר לצורכי עיצוב וקיבוץ: <div> (בלוק שתופס שורה שלמה) ו-<span> (מקטע קטן בתוך שורה). נשתמש בהם הרבה כשנלמד CSS ו-JavaScript.

✅ טיפ

אם יש אלמנט סמנטי שמתאים (כמו <nav> לתפריט), עדיף להשתמש בו. משתמשים ב-div רק כשאין אלמנט מתאים יותר.

HTML הוא לא שפת תכנות

כדי ששפה תיחשב שפת תכנות (Programming Language), היא צריכה לתמוך ב:

  1. משתנים (Variables)
  2. תנאים (if)
  3. לולאות (loops)
  4. הגדרת פונקציות

ל-HTML אין אף אחד מאלה, ולכן היא שפת סימון ולא שפת תכנות. את החלק ה"חכם" והדינמי נוסיף בעזרת JavaScript, ואת העיצוב בעזרת CSS. שלושתם עובדים יחד: HTML לתוכן, CSS לעיצוב, JavaScript להתנהגות.

מגרש משחקים חופשי

עכשיו תורך. שלב כותרות, פסקאות, רשימות וקישורים. שנה את הקוד ותראה מיד את התוצאה. אם התבלבלת, לחץ "אפס".

בדיקה עצמית

ענה על השאלות. אחרי כל תשובה תקבל הסבר. בסוף תראה את הציון שלך.
