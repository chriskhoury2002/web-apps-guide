JavaScript ו-DOM (מבוא)

HTML נותן תוכן ו-CSS נותן עיצוב, אבל הדף עדיין "מת". JavaScript הוא מה שהופך אותו לחי: מגיב ללחיצות, משנה טקסט, מוסיף ומסיר אלמנטים. כדי לעשות את זה, JS עובד מול ה-DOM, ייצוג הדף כעץ של אובייקטים בזיכרון.

JavaScript, שפת הדפדפן

JavaScript (JS) היא שפת התכנות היחידה שכל הדפדפנים יודעים להריץ. כמה עובדות חשובות עליה:

  • יש לה תחביר דמוי-C (כמו Java), עם סוגריים מסולסלים ונקודה-פסיק.
  • היא מפורשת (interpreted), הדפדפן מריץ אותה ישירות בלי הידור מראש לקובץ הרצה (בפועל המנועים המודרניים גם מהדרים אותה תוך כדי ריצה, JIT).
  • היא loosely typed, טיפוס המשתנה נקבע לפי הערך שמכניסים אליו.

כדי לשלב JS בדף, שמים אותו בתוך תגית <script> בכל מקום ב-HTML. שלא כמו HTML, JS היא כן שפת תכנות מלאה, עם משתנים, תנאים, לולאות ופונקציות.

שתי נקודות שהופכות את הכל לאפשרי:

DOM: הדף כעץ אובייקטים

DOM (Document Object Model) הוא ייצוג הדף כעץ של אובייקטים בזיכרון. הדפדפן בונה אותו כשהוא קורא את ה-HTML, ומשתמש בו כדי לצייר את הדף. חשוב להבין: ה-DOM הוא "מסמך חי" בזיכרון, וכל שינוי שנעשה בו דרך JavaScript משתקף מיד על המסך.
