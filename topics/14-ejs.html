EJS תבניות

לפעמים רוצים HTML שמשתנה לפי נתונים: רשימת מוצרים, שם המשתמש המחובר. מנוע תבניות כמו EJS מאפשר לשלב קוד בתוך HTML ולייצר תוכן דינמי, כשהעיבוד נעשה בשרת והלקוח מקבל רק HTML מוכן.

הבעיה: HTML דינמי

לפעמים צריך שינויים תכנותיים קטנים בתוך קובצי HTML. הפתרון: מנועי תבניות (template engines). Express תומך בכמה: Pug, EJS, Mustache, Handlebars ואחרים. נלמד את EJS, שפופולרי ונוח למי שרוצה תבניות מינימליות עם JavaScript רגיל.

הרעיון המרכזי

מפתחים front-end סטטי (HTML+CSS+JS), ובכל מקום שהתוכן צריך להיות דינמי, שמים קוד צד-שרת שמייצר אותו. חשוב: הקוד רץ בשרת, הוא לא נשלח ללקוח. הלקוח מקבל רק את ה-HTML עם הנתונים הדינמיים.

💡 למה זה חשוב

כך הלוגיקה והנתונים הרגישים נשארים בשרת. הלקוח רואה רק HTML מוגמר, בלי הקוד שיצר אותו ובלי גישה למסד הנתונים.

התקנה והגדרה

שלושה צעדים כדי להתחיל עם EJS:

Scriptlets

בתוך קובץ .ejs משלבים קוד עם תגיות מיוחדות:

  • <% %>, מריץ JavaScript בלי פלט (למשל if, לולאה).
  • <%= %>, מדפיס את הערך לתוך ה-HTML.
