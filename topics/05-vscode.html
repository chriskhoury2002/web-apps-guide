סביבת פיתוח: VS Code

כלי טוב חוסך המון זמן. בנושא הזה נכיר את VS Code, את התוספים שכדאי להתקין, את Emmet שכותב HTML במהירות, ואת קיצורי הדרך שיהפכו אתכם למהירים יותר.

עורכי קוד

יש הרבה עורכים לפיתוח ואב: VS Code, Brackets, Atom, Sublime Text, Notepad++, NetBeans, PHPStorm ועוד. הפופולרי ביותר היום, וזה שנשתמש בו, הוא Visual Studio Code (VS Code) של מיקרוסופט. הוא חינמי, מהיר, וניתן להרחבה בעזרת תוספים (extensions).

תוספים מומלצים

הכוח האמיתי של VS Code הוא בתוספים. הנה החשובים:

שווה להפעיל את "Format on Save" בהגדרות. ככה בכל שמירה הקוד מסתדר לבד, ואתם לא מבזבזים זמן על רווחים והזחות.

כלי AI לקוד

כיום יש תוספי בינה מלאכותית שמשלימים קוד ומציעים פתרונות תוך כדי כתיבה, למשל GitHub Copilot ו-Tabnine. הם עוזרים לכתוב מהר יותר, אבל חשוב להבין את הקוד שהם מציעים ולא להעתיק בעיוורון.

Emmet: לכתוב HTML במהירות

Emmet מובנה ב-VS Code ומאפשר לכתוב HTML שלם בעזרת קיצורים קצרים. כותבים אבּרוויאציה ולוחצים Tab, וזה מתרחב ל-HTML מלא. הכללים העיקריים:

  • > לאלמנט בן (child). למשל ul>li
  • * לשכפול. למשל li*3
  • .class ו-#id. למשל div#app.box
  • + לאלמנטים באותה רמה (siblings). למשל input+button
  • $ למספור. למשל li#item$*3

נסה בעצמך. הקלד אבּרוויאציה או לחץ על אחת הדוגמאות, וראה מיד את ה-HTML שנוצר:

קיצורמה עושה

Alt + Shift + ↓שכפול השורה למטה
